Why Kindows exists. I mean will exist. GAH, too stressful. Anyway…
Linux. It’s broadly used. But it’s also a boundary.
Windows has the same problem. Linux apps don’t run on Windows. Windows apps don’t run on Linux. Vice versa, always.
Using the power of Proton and KDE, Kindows aims to make that problem much smaller.

Introduction to Kindows :)
Kindows is an open‑source operating system built on KDE.
It’s my little experiment in blurring the lines between worlds that normally don’t talk to each other — taking the power and flexibility of Linux, the friendliness people expect from mainstream desktops, and smashing them together into something that (hopefully) feels simple, fun, and usable.
